# 🌱 VitroVision — Expert Assessment Rubric (One-Pager)

> **สำหรับผู้ประเมิน (ครู/ผู้เชี่ยวชาญ TC)** · ให้คะแนนจากภาพ 1 ภาพ/ขวด
> *Capsicum frutescens* cv.'พริกจินดา' เพาะเลี้ยงในขวดแยมแก้ว 125 mL · sync กับ methods §4.1
> **ทุกคนได้ rubric ชุดเดียวกันก่อนเริ่ม — ห้ามปรึกษากันระหว่างให้คะแนน (blind)**

---

## บันทึก 3 อย่างต่อขวด: **Axis A + Axis B + Flag**

### 🅰️ Axis A — Developmental phase (เลือก 1 ข้อ, นับโครงสร้างได้แบบ objective)

| Phase | เกณฑ์ |
|---|---|
| **Intensive growth** | งอกแล้ว + กำลังสร้างใบ/ยอดเร็ว (ช่วงต้นของ sigmoid) |
| **Slow growth** | การเจริญชะลอ เข้าใกล้ plateau |
| **Senescent / dying-off** | เหลือง / น้ำตาล / ตาย |

---

### 🅱️ Axis B — Vigor quality (ให้คะแนน 1–5 แบบ **holistic** = ดูภาพรวม ไม่ใช่วัด feature เดียว)

> ตัดสิน "ความสมบูรณ์โดยรวม" จากหลาย cue พร้อมกัน: สี + ทรงพุ่ม + จำนวนยอด + ความสมบูรณ์
> ⚠️ **อย่าให้คะแนนจาก "% พื้นที่เขียว" อย่างเดียว** — ต้องเป็นความรู้สึกภาพรวม (gestalt)

| Grade | นิยาม holistic | cue ประกอบ |
|---|---|---|
| **1** | ตาย / ไม่งอก น้ำตาลเด่น ไม่มีส่วนเขียวมีชีวิต | — |
| **2** | อ่อนแอ งอกแต่ชะงัก เหลือง/ซีด เขียวน้อย | ไม่มียอดชัด / ยอดเดียวอ่อน |
| **3** | ปานกลาง เขียวบางส่วน เริ่มตั้งตัว | มี ≥1 ยอด ใบเริ่มกาง |
| **4** | ดี แข็งแรงชัด เขียวเข้ม ทรงสมบูรณ์ | ยอดชัดเจน ใบกางดี |
| **5** | ดีมาก สมบูรณ์เด่น พุ่มเขียวเข้มแน่น | หลายยอด / พุ่มแน่น แข็งแรงเต็มที่ |

> 🔶 **โซนที่ต้องเคาะให้ชัด — เส้นแบ่ง Grade 3 vs 4:**
> - **3 = "เริ่มตั้งตัว"** ใบเริ่มกางแต่ทรงยังไม่สมบูรณ์ / เขียวยังไม่เข้มทั่ว
> - **4 = "แข็งแรงชัด"** ใบกางดีแล้ว เขียวเข้ม ทรงดูสมบูรณ์ครบ
> - 👉 *ถ้าลังเลระหว่าง 3/4:* ดู "ทรงพุ่ม + ความเข้มของเขียว" เป็นตัวตัด — ครบ+เข้ม = 4, ยังขาด = 3
> - **(ช่องนี้รอครู TC review + เติม anchor image จริงตอน calibration round)**

---

### 🚩 Flag — Hyperhydric (binary: มี / ไม่มี — บันทึกแยกจาก Axis B)

อาการ vitrification: **ใบใส / ฉ่ำน้ำ / บวม / ซีดผิดปกติ**
⚠️ **ไม่ใช่แค่ vigor ต่ำ** — เป็น abnormal phenotype คนละแกน → **อย่ายุบเป็น Grade 1** ให้ติ๊ก flag แยก
(ต้นโตช้าธรรมดา ≠ hyperhydric)

---

## 📋 วิธีให้คะแนน (สำคัญ)

1. ดูภาพทีละภาพ ตามลำดับในชีต (ลำดับถูกสุ่มแล้ว — ภาพต่างสูตรปนกัน **โดยตั้งใจ**)
2. กรอก **Axis A (phase) + Axis B (1–5) + hyperhydric (Y/N)** ลง scoring sheet
3. **ห้ามดู:** สูตรอาหาร / bottle_id / คะแนน CV / คะแนนของผู้ประเมินคนอื่น
4. ถ้าภาพไม่ชัด/ตัดสินไม่ได้ → กรอกในช่อง notes ไว้ (อย่าเดามั่ว)
5. ทำคนเดียว ไม่ปรึกษากันระหว่างทำ
