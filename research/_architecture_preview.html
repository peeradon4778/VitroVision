📈 ANALYSIS & VALIDATION — NEW v2

not started

M1 — Inter-rater Reliability

  • ICC(2,1) absolute agreement ≥ 0.75
  • Quadratic-weighted κ vigor-rubric ≥ 0.61
  • Cohen's κ binary (hyperhydric)
  • Krippendorff α (rater ≥ 3)

M2 — CV vs Expert

  • Spearman ρ ≥ 0.80 + 95% CI bootstrap
  • Bland-Altman bias + LoA

M3 — Treatment Comparison

  • ART-ANOVA + ART-C post-hoc (ไม่ใช่ Dunn)
  • ε²p effect size (ไม่ใช่ η²)
  • 3 planned contrasts: A/B/C · C/D · A/E

M4 — Growth Curve

  • Gompertz/ขวด: K · k · tm · AUC
  • LMM treatment comparison on K/k/tm

M5 — Survival/Contamination

  • KM + permutation log-rank
  • CIF ถ้า competing events

⛔ Guardrails — ข้อเท็จจริงที่ห้ามพลาด

ห้ามอ้างผิด / ห้ามทำผิด

  • κ=0.6274 ใน metrics.json = smoke-test บนภาพมั่ว — ห้ามอ้างเป็น metric จริง
  • classical CV (green%/HSV/ExG) ≠ AI — เป็นคณิตศาสตร์ภาพ; AI จริง = EfficientNet/DINOv2
  • VLM ห้ามออกตัวเลข vigor — qualitative description เท่านั้น (Q9)
  • post-hoc = ART-C เท่านั้น — ห้ามใช้ Dunn
  • ห้าม pool data engine เก่า (HSV) กับ engine ใหม่ (MobileSAM) — ต้อง bridge study ก่อน
  • ขวด = experimental unit — ห้ามนับภาพ 20–25 ภาพ/ขวดเป็น n (pseudoreplication)
  • แยก κ vigor-rubric ≠ κ classifier-3class ทุกที่
  • สปีชีส์: Capsicum frutescens Mill. — ไม่ใช่ C. annuum
